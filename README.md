# Billnote（領収書管理アプリ）

# 概要
領収書管理と顧客管理ができるWEBアプリです。
領収書（Excel）のアップロードと顧客名や年月で検索ができます。

# 目的・背景
実家が営んでいる畳屋の領収書管理をもっと簡単にしたいと考え、
シンプルで使いやすい領収書管理兼顧客管理アプリを開発しました。

# 使用技術
Python / FastAPI
HTML / CSS / Javascript
AWS(DynamoDB, S3) / Git

# 使用方法・機能
1⃣　対象年・対象月を選択する。
2⃣　ファイルまたはフォルダーをドラッグ＆ドロップしてアップロードボタンを押す。
・Excelがアップロードされ、Excelから顧客名・住所・電話番号・金額・を抜き出している。
・電話番号で顧客の識別をしており、DynamoDBに同じ電話番号がない場合は新規顧客ID、ある場合は同じ顧客ID、
　Excelに電話番号が入っていない場合は顧客IDに0をつけている。
・ExcelはS3へ保存され、対象年月/抜き出した情報/顧客ID/S3のパスがDynamoDBへ保存される。
3⃣　検索したい年月を選択または顧客名を入れて検索ボタンを押す。
・顧客名は部分一致検索が出来るようになっています。
4⃣　検索内容をもとにDynamoDBに保存されていたデータが一覧として表示されます。
・顧客IDがリンクになっておりクリックすると顧客IDで検索されます。
・操作欄の保存ボタンを押すと領収書(Excel)がダウンロードできます。
